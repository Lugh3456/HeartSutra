# Heart Sutra · 般若波罗蜜多心经

A personal bilingual study site for the Heart Sutra, one of nine scripture mini sites under [The Dharma Gate](https://lugh3456.github.io/DharmaGate/) portal.

## What it is

A line-by-line study companion for the Heart Sutra, divided into three sections: Opening (序分), Main Body (正宗分), and Mantra (咒语). Each section presents the full Chinese text with speech synthesis, followed by explanation cards with short Chinese explanations and expandable detailed English commentary.

## Features

- Bilingual Chinese/English throughout
- Default view shows Chinese explanations only (concept-focused, accessible register)
- Expandable "Details" toggle reveals English translation + full English commentary
- Bilingual summary at the end of each section (Chinese default, expandable English)
- Speech synthesis for full Chinese text (Web Speech API)
- Portal bar linking back to The Dharma Gate hub
- Mobile-optimised, responsive layout

## Tech stack

- Pure HTML + external CSS, no frameworks
- CSS design tokens (red/saffron colour scheme)
- Google Fonts: Noto Serif SC + Noto Sans SC
- No build step required

## Local preview

Open `index.html` directly in a browser.
